
# 评审 Demo 脚本（90 秒）

1. 抖音视频内点开小程序卡片（0-10s）
   - 话术：这条视频的“南瓜鸡肉粥”想学？点卡片进来。

2. 落地页=菜谱详情（10-25s）
   - 展示食材清单与步骤，强调“一键买齐”和“AI定制”。

3. AI定制（25-50s）
   - 填：猫 / 英短 / 12个月 / 3.8kg / 无过敏
   - 模型返回：克数与注意事项（可朗读关键点）。

4. 一键购买（50-70s）
   - 点击跳转第三方平台购物车（展示已添加的食材）。

5. 回跳收藏（70-90s）
   - 返回小程序弹出“收藏菜谱/编辑宠物档案”。
   - 总结：内容→AI→下单的完整闭环，适合规模化联动品牌与食材平台。
